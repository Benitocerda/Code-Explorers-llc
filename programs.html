Programs

In-school, after-school, and summer options. We tailor pacing and tools by grade band.

Intro to Coding

Grades: 3–12 • Length: 8–10 weeks • Pathway: Blocks → Python/JS

  • Concepts: sequencing, loops, conditionals, functions.
  • Outcomes: 3–5 shipped mini projects + final showcase.
  • Tools: age-appropriate (e.g., Scratch → Python/JS).

Robotics

Grades: 4–12 • Length: 8–12 weeks • Pathway: Build → Wire → Code

  • Sensors, motors, control flow, debugging.
  • Team challenges and demo day.

Creative Tech

Grades: 5–12 • Length: 6–8 weeks • Focus: Games, art, micro-electronics

  • Game jams, pixel art, simple electronics.
  • Portfolio-ready artifacts.

Delivery Models

  • After-School (1–2x/week, 60–90 min)
  • In-School (push-in, stations, or full class)
  • Summer Intensives (daily, 1–2 weeks)
  • PD & Kits (turnkey for teachers)
